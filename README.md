Учебный проект UI-тестов на Python + Playwright Данные тесты проверяеют отображение элементов и работу функционала блока Отели в сервисе Smartway.

Структура проекта:

Папка pages/ - Page Object классы

pages/start_page.py - Базовый класс для всех страниц

pages/login_page.py - Страница авторизации

pages/region_page.py - Страница выдачи после поиска

pages/hotel_search_page.py - Страница строки поиска

pages/smartdesk_page.py - Страница смартдеск

Папка tests/ - Тесты

test/test_hotels.py - Тесты для отелей

conftest.py - Фикстуры pytest

Правила:

Каждая страница имеет свой класс локаторов

Локаторы группируются по страницам для удобства поддержки

Методы проверок должны содержать expect()

Все тесты должны называться с test_

Условия доля локального запуска тестов:

Установить python 3.12

В проекте установить зависимости из файла requirements.txt (pip install -r requirements.txt)

Команда для запуска в безголовом режиме: python -m pytest tests/test_hotels.py

Команда для запуска с визуализацией: python -m pytest tests/test_hotels.py --headed --slowmo 1000

Перечень тестов:

Положительные сценарии:

Проверка на успешный логни

Проверка активности кнопки поиска на странице поиска отелей

Проверка успешного поиска и наличии выдачи отелей

Проверка на наличие рейтов на выдаче с бесплатной отменой

Негативные сценарии:

Проверка на невалидный логин

Проверка на отсутствие выдачи с бесплатной отменой рейта
